▶ Commands

Commands take the current selection as an argument.?

The smart insertion will by default auto-detect the first empty brackets (curly {} or square []) in the command as where to insert the selected text and keep that selection. For example, the Alt-\ command by default takes *selection* into \frac{*selection*}{}. Both of these behaviours can be customized separately. The selection insertion point can be specified with %.%, which is replaced with the selected text, as in the Alt-( default command. The desired end selection can be bounded with %|%, or for an empty end selection (i.e. just an end cursor position), a single %|%.
